How does javascript Works?

Everything in javascript happens inside an "Execution context". Execution Context has two components in it, one is memory component and another one is code component. In memory component all the variable and function is stored as a key value pair it is known as variable environment. In code component , whole code is executed one line at a time in a order once a line is excueted it then excuete the next line of code which is known as thread of execution And that's why Javascript is a asynchronous single threaded language.I

Local storage vs Session storage

In local storage we can store any kind of data which we need to access even after the browser is closed and here data is stored as key value pair and it has no expiration time. But in Session storage we store the for any particular session , so when browser is closed data will be lost as well and it has expiration time. In session storage data is stored as a key value pair like local storage.
